French Language Pack for Flarum
===============================

GitHub Repository: https://github.com/maelsoucaze/flarum

French language pack to localize the Flarum forum software and its official extensions.

Installation
------------

Flarum Documentation: http://flarum.org/docs/languages/

1. Unpack this .zip archive.
2. Log in to your server via SSH, FTP, or your provider's control panel.
3. Browse to the extensions/ directory of your Flarum installation.
4. Create a new subdirectory named french/ (without the slash punctuation mark at the end).
5. Upload everything in the unpacked folder to the subdirectory you just created.
6. Use your browser to navigate to the extensions page of the administration interface.
7. Locate the French language pack and enable it.
