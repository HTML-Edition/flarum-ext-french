French Language Pack for Flarum
===============================

See: https://github.com/maelsoucaze/flarum

French language pack to localize the Flarum forum software
and its official extensions.

Installation
------------

See: http://flarum.org/docs/languages/

All the unpacked content of this archive must be uploaded
into a french/ subdirectory that you have to create in
the extensions/ directory.
